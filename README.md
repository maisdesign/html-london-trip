# Il mio viaggio a Londra

Progetto web statico in HTML che mostra un riepilogo di un viaggio a Londra: spese, cose da fare, informazioni utili e una galleria di foto.

## Contenuto

- Intestazione con titolo e immagini principali.
- Tabella delle spese del viaggio.
- Lista delle attività (con indicatori di completamento).
- Informazioni utili: numeri di emergenza, hotel e voli.
- Sezione foto con immagini esterne.

## Struttura del progetto

```
.
├─ index.html
└─ img/
   ├─ pexels-bill-emrich-230794-0.jpg
   ├─ pexels-bill-emrich-230794-1.jpg
   └─ pexels-bill-emrich-230794-2.jpg
```

## Come vedere il sito

1. Apri `index.html` nel browser.
2. Mantieni la cartella `img/` nella stessa posizione per caricare le immagini locali.

## Note

- Il contenuto della pagina è in italiano.
- Le foto della sezione finale sono caricate da URL esterni.
